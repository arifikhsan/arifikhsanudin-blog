# Personal Blog

## Prerequested

- NodeJS
- Git

## Install

1. Clone repo

```shell

git clone git@github.com:arifikhsanudin-blog.git

```

2. Install dependencies

```shell

yarn

```

## Run Development

```shell

yarn dev

```

## Run Production

```shell

yarn prod

```
